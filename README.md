## ✨ Project Highlights

* **Glassmorphism Design:** A modern, frosted-glass UI built entirely from scratch without relying on heavy frameworks.

* **Interactive Elements:** Smooth hover animations and integrated FontAwesome iconography for a polished user experience.
* **Custom Typography:** Implementation of Google Fonts to create a strong visual hierarchy.

## 💻 How to Run Locally

Since this project is built with pure HTML and CSS, no complex setup or server is required!
1. Clone this repository to your local machine.
2. Open the project folder.
3. Double-click the `index.html` file to run it instantly in your web browser.
